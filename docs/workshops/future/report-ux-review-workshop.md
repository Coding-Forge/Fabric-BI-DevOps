# Report UX Review Workshop

Placeholder for a future workshop focused on report usability, accessibility, navigation, layout, and mobile-readiness review.

## Intended outcomes

- Create a repeatable UX review checklist.
- Assess page density, titles, tooltips, theme use, navigation, accessibility, and mobile layout.
- Feed findings into PR quality and release readiness reviews.

- PBIP Diff Viewer
