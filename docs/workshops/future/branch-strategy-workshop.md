# Branch Strategy Workshop

Placeholder for a future workshop focused on branch policy, feature workspaces, PR gates, protected branches, and reviewer preview workflows.

## Intended outcomes

- Recommend branch strategy for a Fabric/PBIP team.
- Map branches to shared, personal, and scoped feature workspaces.
- Define required PR reviewers and status checks.
- Identify anti-patterns such as editing directly in shared Dev.
